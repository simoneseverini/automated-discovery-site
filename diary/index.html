Today I was thinking about the Hamiltonian simulation problem. If we define the operator as: $$ H = \sum_{i} \sigma_z^{(i)} \otimes \sigma_z^{(i+1)} $$ We can observe that the evolution $e^{-iHt}$ acts trivially on the subspace. This is a simple test of the rendering.
