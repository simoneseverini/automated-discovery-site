December 26, 2025

A Markov triple is a triple of positive integers $(x,y,z)$ satisfying $$x^2+y^2+z^2 = 3xyz.$$ The integers that occur in Markov triples are the Markov numbers. The usual way to navigate the set of solutions is the Vieta move: if $(x,y,z)$ is a solution, then so is $$(x,y,3xy-z),$$ and similarly by cycling coordinates. Iterating these involutions (and permuting) generates the Markov tree.

Fibonacci numbers are slower creatures: $F_0=0$, $F_1=1$, and $F_{n+1}=F_n+F_{n-1}$.

The question for today is simple: which Markov triples have a Fibonacci number as their largest entry? There is one branch where Fibonacci numbers show up for free: the branch of triples that contain a $1$. This is not an “observation”; it is a quick computation.

If $x=1$, the Markov equation becomes $$y^2+z^2+1 = 3yz.$$ Solving for $z$ gives $$z=\frac{3y\pm\sqrt{5y^2-4}}{2},$$ so integrality forces $5y^2-4$ to be a square (a Pell-type constraint). The resulting positive solutions are exactly $$(1, F_{2n-1}, F_{2n+1}) \qquad (n\ge 1),$$ e.g.

  • $(1,F_1,F_3)=(1,1,2)$
  • $(1,F_3,F_5)=(1,2,5)$
  • $(1,F_5,F_7)=(1,5,13)$

Moreover this branch is generated by a deterministic Vieta step that preserves the $1$: $$(1,a,b)\longmapsto (1,b,3b-a),$$ and the identity $F_{2n+3}=3F_{2n+1}-F_{2n-1}$ makes the Fibonacci indexing explicit. This is the “Fibonacci spine”.

The conjecture is the sharp one, namely, no other Markov triple has a Fibonacci maximum. This is a special case of Markov uniqueness (Frobenius): each Markov number should be the maximal entry of a unique triple (up to permutation).

I added a Colab notebook on GitHub that draws the tree with this spine highlighted. Off the spine, the Vieta update is typically multiplicative ($z'\approx 3xy$ once both $x$ and $y$ grow), so along rays the numbers blow up at a rate that is “exponential in an exponential” (because the logarithms satisfy a Fibonacci-like recursion). Fibonacci numbers, in comparison, stay unreasonably close together.

Figure 1: The Fibonacci spine (red) and the side branches (blue). Once you leave the spine, the magnitudes accelerate so fast that hitting a Fibonacci value starts to feel like a category error.

The Shannon capacity of a graph $G$ is defined as $\Theta(G) = \lim_{d \to \infty} (\alpha(G^d))^{1/d}$. The best lower bound for 7-cycle graph $C_7$ was obtained by finding an independent set of size 367 in the fifth power. This yields: $\Theta(C_7) \geq \sqrt[5]{367} \approx 3.2578.$ See arXiv:1808.07438 for details. Below is a Colab notebook generated with AlphaEvolve. It gives the same result.
